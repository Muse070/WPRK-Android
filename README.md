# WPRK (Android)
Radio &amp; Podcast Streaming App For WPRK, a licensed FCC broadcast station,
The app allows users to connect to live music streams and cycle through
podacasts which are playable as well. 

## TechStack
- Jetpack Compose
- Exoplayer
- Hilt
- Retrofit
- Accompanist
